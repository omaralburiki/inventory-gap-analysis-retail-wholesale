11. Ownership Model
 
| Process Area | Owner | Accountability | Reporting Frequency |
|-------------|-------|---------------|-------------------|
| Inventory Accuracy (per branch) | Branch Manager | Daily variance < 3% | Daily dashboard |
| Barcode Scan Compliance | Warehouse Supervisor | 100% scan rate at dispatch | Real-time via ERP |
| Exception Handling | Operations Lead | Resolution within same shift | Shift report |
| SKU Data Quality | Procurement Manager | Zero active duplicate SKUs | Weekly audit |
| POS ↔ WH Integration Health | IT / ERP Owner | Zero unresolved sync failures | Daily alert |
| KPI Review | General Manager | Trend review + corrective action | Weekly |
