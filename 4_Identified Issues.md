## 4. Identified Issues
 
| # | Issue Area | Description | Estimated Impact |
|---|-----------|-------------|-----------------|
| 1 | Inventory Accuracy | 10–20% variance between system records and physical stock | Financial misstatement, failed fulfillment |
| 2 | Real-Time Visibility | No live inventory tracking during dispatch | Unknown shortages discovered at pick time |
| 3 | Manual Entry | 100% manual data entry — no barcode system | High human error rate, slow processing |
| 4 | System Integration | POS and Warehouse systems not connected | Data silos, delayed and inconsistent updates |
| 5 | Customer Experience | Customer physically walks POS → WH → POS per transaction | Slow, unprofessional, high abandonment risk |
| 6 | Purchasing Decisions | Procurement makes decisions on unreliable system data | Overstock on some SKUs, stockout on others simultaneously |
| 7 | SKU Complexity | Same product from multiple origins registered as separate SKUs | Miscounts, wrong picks, confusion for warehouse staff |
| 8 | Exception Handling | No defined protocol when items are unavailable | Each exception handled differently by different staff |
