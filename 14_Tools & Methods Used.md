## 14. Tools & Methods Used
 
| Tool / Method | Purpose |
|-------------|---------|
| BPMN Process Mapping (draw.io) | AS-IS and TO-BE workflow documentation |
| 5-Whys Root Cause Analysis | Structured problem diagnosis |
| Gap Analysis Framework | Structured comparison of current vs. required state |
| Mermaid Diagrams (GitHub) | Process flow visualization embedded in documentation |
| Risk Register | Proactive risk identification with mitigation plans |
| Stakeholder Analysis | Change resistance mapping and ownership model |
| KPI Framework Design | Before/after measurement structure |
| Excel / Google Sheets | Gap table, KPI tracking, roadmap |
