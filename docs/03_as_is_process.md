## AS-IS Process

POS Staff → Creates Draft Order
Customer → Goes to Warehouse
Warehouse → Manual picking + cross out unavailable items
Customer → Returns to POS
POS → Issues invoice

* No real-time stock visibility
* Inventory updated with delays
* Paper-based workflow
